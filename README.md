# Сезоны — прототип управленческого дашборда

Статичный прототип на чистом HTML/CSS/JS. Данные в макетах случайные.

## Страницы
- `index.html` — дашборд (главный экран)
- `foodcost.html` — фудкост по подразделениям
- `app/styles.css` — общие стили обеих страниц

## Деплой на GitHub Pages
1. Залить всё содержимое этой папки в корень репозитория (сохранив папку `app/` и файл `.nojekyll`).
2. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
3. Сайт поднимется по https://USERNAME.github.io/REPO/

Важно: имена файлов чувствительны к регистру на сервере — не переименовывать.

# Сезоны — прототип управленческого дашборда

Статичный прототип (HTML/CSS/JS). Данные в макетах случайные.

## Файлы
- index.html        — дашборд
- foodcost.html     — фудкост
- app/base.css      — ОБЩИЕ стили обеих страниц (менять осторожно — влияет на всё)
- app/dashboard.css — стили только дашборда
- app/foodcost.css  — стили только фудкоста (правь свободно, дашборд не заденешь)

## Как править стили
- Меняешь фудкост → app/foodcost.css. Дашборд эту страницу не грузит, сломать его нельзя.
- Меняешь дашборд → app/dashboard.css.
- Общую оболочку (сайдбар, карточки, палитра) → app/base.css, сознательно, затрагивает обе.
Перед деплоем: git diff покажет, в каком файле изменения — если правил фудкост, а diff в base.css, это сигнал.

## Локальный запуск (двойной клик НЕ работает — внешний CSS)
    python3 -m http.server 8000
затем http://localhost:8000/

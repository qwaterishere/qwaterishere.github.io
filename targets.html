Инвентаризация

[05]
допустимая недостача по итогам месячной инвентаризации
факта нет — домен фазы 2
◯ TARGETS / МЕСЯЦ Σ ДНЕЙ = ПЛАНУ ✓
Решения: план выручки задаётся ПО ДНЯМ (сумма месяца × профиль недели, дни можно будет докрутить вручную); наследования из прошлого месяца НЕТ — каждый месяц заполняется заново; права на изменение целей — вопрос открыт (упирается в роли, карточка №11). Проценты пересчитываются от плана выручки — зелёные подписи. Цели уезжают: выручка → дашборд (plan дня на графике, трек прогноза), фудкост → foodcost goal, списания/комплименты → losses, инвентаризация — блок фазы 2.

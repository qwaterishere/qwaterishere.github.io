Фудкост — путь каждого элемента

Как каждый блок страницы «Фудкост» получает данные: строка БД → функция-сборщик → схема → HTTP → фронт-мапер → пиксель. Числа обезличены (заведение «Веранда», ₽).
/foodcost

Общий принцип: бэкенд — факты, фронт — представление

Сервис build_food_cost() отдаёт по каждому разрезу три сырых факта (revenue, cost, revenueWithCost). Проценты фудкоста, доли, LfL и подписи считает фронт в foodcost.mapper.ts. Ниже — каждый элемент страницы отдельно; зелёное = бэкенд (Python), фиолетовое = фронт (Angular, зона коллеги).
